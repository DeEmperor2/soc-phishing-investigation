## Case Studies

### Case 001 - Phishing Email Analysis

**Summary**

A phishing email claiming to contain failed email deliveries was analyzed. The investigation involved:

- Email header analysis
- SPF, DKIM, and DMARC verification
- IOC extraction
- URL investigation
- VirusTotal verification
- MITRE ATT&CK mapping

**Findings**

- Sender impersonation attempt
- Phishing infrastructure identified
- Malicious domain detected by multiple security vendors
- Embedded phishing URL designed to steal credentials

**Artifacts**

- Sample email (.eml)
- Analysis report
- IOC table
- MITRE ATT&CK mapping
- Evidence screenshots
